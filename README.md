# gis
针对特定地图框架方法进行封装或定制化开发，并提供常用的地图数据处理工具，实现地图渲染以及地图交互。
## 使用方法

* 安装依赖

```
npm install
```

* 运行VitePress查看手册

```
npm run docs:dev
```
